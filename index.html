4.1- Quais os comandos para configurar usuário e email no git?

git config --global user.name "w3schools-test"
e git config --global user.email "test@w3schools.com"

4.2- Qual a diferença entre git e github?

Git it é um software VCS mais para quem trabalhar e um projeto individual e o GitHub para quem trabalha em um projeto com varios colaboradores

4.3- O que faz o comando commit?

O commit comenta cada alteração feita no projeto assim sabem o que foi feito e quem altero

4.4- O que faz o comando push?

Envia uma alterção do git local para um repositorio no github
